# Learning Playwright 🚀
A hands-on Playwright learning journey documenting concepts, examples, assignments, and practical exercises from beginner to advanced topics.
This repository accompanies my Medium blog series ["Learning Playwright"](https://medium.com/@architatotla/list/learning-playwright-121c6c8be851), where I share what I'm learning, code examples, and assignments for fellow automation enthusiasts.

## Who is this for?
- Beginners starting with Playwright
- Manual testers moving into automation
- QA Engineers exploring modern web automation
- Anyone looking for practical Playwright examples

## How to follow along
Step 1 : Clone the repository
`git clone https://github.com/archita-at/learning-playwright`

Step 2 : Checkout the corresponding branch to follow along with a specific blog
`git checkout day-1`
NOTE: For answers to the coding assignment, the branch name would add -assignments after the corresponding branch
For example:
`git checkout day-3-assignment`
